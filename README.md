# Personal CV / Resume

## Overview

This project is my CV / Resume site.

## Stack
- Github Pages for hosting

## Getting Started
### Prerequisites
- Git
- Docker
### Installation / Run locally 
Once installed, to create new site and run it locally
```sh
docker-compose up
```
